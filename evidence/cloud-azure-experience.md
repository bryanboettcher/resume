# Cloud & Azure Experience

## Overview

Substantial Azure experience spanning Call-Trader/Madera (Jun 2024 – Oct 2025) and Taylor Summit Consulting (2023 – Oct 2025). Taylor Summit work was entirely proprietary and cannot be examined directly. Some Madera Azure components may exist in branch snapshots.

## Azure Services Used

- **Identity** — Azure AD / Entra ID authentication
- **App Service** — Web application hosting
- **Container Apps** — Both Linux and Windows container workloads
- **Azure SQL** — Managed SQL Server instances
- **Service Bus** — Messaging and queuing
- **Blob Storage** — Object storage
- **SMTP Services** — Email delivery
- **Communications Services** — Telephony and SMS integration
- **AI Services** — Hosted AI model consumption
- **Networking** — VNets, NSGs, and related networking components

## Patterns

- Containerization (both greenfield and lift-and-shift)
- Azure-native service integration
- Cloud networking and security configuration
