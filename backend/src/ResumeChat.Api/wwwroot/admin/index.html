Bulk Push from Local Checkout

The browser cannot read local files directly. Use this shell script from your git checkout to push all corpus files to the API: 

#!/usr/bin/env bash
# Run from the root of your git checkout
API_URL="https://resume-chat.mallcop.dev/api/admin/corpus"
API_KEY="YOUR_KEY"

find evidence projects links -name '*.md' | while read -r file; do
  echo "Pushing: $file"
  curl -s -X POST "$API_URL" \
    -H "X-Api-Key: $API_KEY" \
    -H "Content-Type: application/json" \
    -d "$(jq -n --arg path "$file" --rawfile content "$file" \
      '{sourcePath: $path, content: $content, embed: false}')"
  echo
done
